Technical Markdown in. A finished paper out — and your equations survive the trip to Word.

Colophon is a private document studio for Markdown with math. Write it — or paste it straight from ChatGPT or Claude, \(…\) delimiters and all — and it renders as a typeset page. Work in real folders on your disk, then export paper-quality PDF, standalone HTML, or a Word file whose equations arrive native and editable, not as images. Nothing you write ever leaves your machine.

Private by construction

One local HTML file. No server calls, no account, no telemetry — claims you can check, because the application code ships unminified. Remote images in documents are blocked by default, so a pasted document can't phone home either. Exported HTML carries a strict CSP and zero scripts.

Math that survives Word

The #1 way technical documents break: equations turn into images (or garbage) on the way to .docx. Colophon converts KaTeX → MathML → OMML so Word gets real equations. Verified by opening exports in actual Microsoft Word — which caught three library bugs that structural XML checks passed (three library bugs were caught this way that structural checks had passed).

Your files, your folders

Link a folder and it becomes a project: documents are plain .md files on your disk, figures live in figs/, everything works in git, Dropbox, or a plain backup. No import, no lock-in — the offline-Overleaf model. (Browser storage remains as a zero-setup fallback.)

How it works

  1. Download one file. Open colophon.html in any modern browser — from your Downloads folder, a USB stick, or an air-gapped machine.
  2. Write, or paste. Markdown with LaTeX math renders as you type — including the \(…\)/\[…\] delimiters LLMs emit. Open a .tex file and Colophon offers a Markdown conversion, math kept intact. Link a folder to work on real files; paste images and they land in figs/.
  3. Export. PDF (print-quality page breaks), standalone HTML (script-free, CSP-locked), LaTeX, Markdown — or Word, with equations your collaborators can keep editing. Autosave, version history, BibTeX citations, Mermaid diagrams, footnotes, callouts included.

Honest FAQ

Why is the file ~7 MB?
KaTeX's math fonts are embedded so math renders offline, and the Mermaid diagram engine ships as an inert blob that isn't even parsed until a document actually uses a diagram. It's a one-time download that then works forever, anywhere.
Is it really offline?
Yes — the page makes zero network requests. Remote images in your documents are blocked by default (per-document opt-in), so a pasted document can't phone home either.
Where do my documents live?
Preferably in a folder you pick: real .md files on your disk that any other tool can read (folder linking needs a Chromium browser). Without a folder, documents live in the browser's local storage — exportable to .md anytime.
What about the messy Markdown LLMs produce?
The renderer meets it where it is: \(…\) and \[…\] math render directly, GitHub-style callouts and task lists work, and LaTeX pastes get a conversion offer. There's deliberately no auto-"repair" feature — earlier versions had one, and it couldn't keep up with real-world paste variants, so it was removed rather than shipped half-working.
Why a browser app and not a native one?
The single file is the zero-install trial. If there's real demand for file associations, one-click PDF, and a menu bar, a native app is the roadmap — the engine is built to be wrapped. Tell us if you'd want that.
What's with the name?
A colophon is the finishing mark of a well-made book — the note at the end that says how it was made. Fitting, for the finishing touch on your documents.
